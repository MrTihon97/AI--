# AI-тренажёр для менеджеров по продажам (MVP)

Фронтенд на **React + Vite + TypeScript + Tailwind CSS**.  
Бэкенда нет — Smart Mock API + `localStorage`. Интерфейс ощущается как живой продукт.

## Как смотреть демо за 90 секунд

1. Дашборд → клик по этапу (подсказка) → toggle задачи плана.  
2. «Начать ролёвку» → **Марина** → карточка → чат.  
3. Напишите: привет → вопрос про потери заявок → «сколько стоит?» → предложите демо.  
4. Смотрите «печатает…» и стриминг ответа.  
5. «Завершить» → разбор с **вашими** цитатами → «На главную».  
6. Стрик/история обновились → клик по записи истории.

Скринкаст 2–3 мин: тот же сценарий без refresh на `/feedback`.

## Ключевые решения (почему так)

1. **Api-слой (`src/services/api.ts`)** — UI не знает про JSON. Подмена на LLM/бэк = одна функция `getSmartReply`.  
2. **Smart Mock вместо линейного скрипта** — иначе чат ломается на первой «не той» реплике; банк + интенты держат ощущение диалога.  
3. **Persona overlays 90%** — общий банк большой, но голос Марины/Артёма приоритетнее, чтобы звучало про стоматологию.  
4. **Динамический разбор по `intentLog`** — не только шаблон из мока: цена до discovery штрафуется, цитаты из чата.  
5. **Стриминг ответа** — бонус из ТЗ; выбран вместо дашборда РОПа: сильнее бьёт в «живость» чата.

## Что сделано

- 3 экрана: Dashboard / Roleplay / Feedback  
- Delay ~300 ms в api (имитация сети) + скелетоны  
- 1000+ реплик, интенты, anti-repeat, лёгкая память диалога (цена уже была / patience)  
- Стриминг + typing по длине реплики  
- Автосохранение прогресса на разборе (`localStorage`)  
- План, история с полным feedback, подсказки этапов, сброс прогресса  
- 2 сегмента клиентов: клиника-одиночка и сеть  

## Что осознанно вырезано

| Вырезано | Почему |
|----------|--------|
| Реальные LLM / бэк / БД | Условие ТЗ |
| База знаний | Вне скоупа тренажёра |
| Auth, оплаты, голос | Не нужны для E2E |
| Дашборд РОПа | Бонус; выбрали стриминг + 2-й клиент |
| Автотесты E2E | В ТЗ «не нужно»; есть smoke интентов |

## Слой моков

```
mock-data.json          → профиль, клиенты, шаблоны разбора, история
dialogue-bank*.json     → интенты и пулы реплик
persona-overlays.json   → голос Марины / Артёма
api.ts                  → единственная точка входа для UI
intentRouter.ts         → классификация + ответ
feedbackScore.ts        → динамические баллы
localStorage v2         → прогресс между сессиями
```

## Как применял ИИ
