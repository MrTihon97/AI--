## Деплой Vercel

1. Создать **новый** GitHub repo, залить **только** содержимое `ai-trenazher`:

```bash
cd ai-trenazher
git remote add origin https://github.com/<you>/<repo>.git
git push -u origin master
```

2. Vercel → Import → этот repo (Root = `.`)  
3. Build: `npm run build` · Output: `dist`  
4. Ссылку вставить в README (Демо + Репозиторий)
